# Installing PsychoPy on Apple Silicon Macs

[PsychoPy](https://www.psychopy.org) (Standalone) runs currently on Apple Silicon Macs through [Rosetta 2](https://en.wikipedia.org/wiki/Rosetta_(software)). I've created instructions to install PsychoPy (Standalone) from source so that it runs natively on Apple Silicon.

Installation instructions for [latest version of PsychoPy (2026.1.3)](v2026.1.3.md). For previous versions of PsychoPy check the folder `older_versions`.

Latest version works natively on Apple Silicon Macs. Unfortunately, a number of components are not working because the free version of the Psychtoolbox library used by PsychoPy is not available for arm64 (Apple Silicon).
